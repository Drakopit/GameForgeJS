WorldEditor agora e uma ferramenta externa

A GameForgeJS continua sendo JavaScript puro. O editor desktop fica separado em C:\Projects\GameForgeJsEditor, podendo abrir qualquer pasta, detectar projetos existentes ou criar uma estrutura nova sem depender de gameforge.editor.json.

Para rodar: abra um terminal em C:\Projects\GameForgeJsEditor e use npm run dev. Dentro do editor, use Escolher pasta para selecionar qualquer jogo/projeto.
